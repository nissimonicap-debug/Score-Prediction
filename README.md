# Score-Prediction

## Problem Statement
Predict student exam score using study hours, sleep hours and attendance.

## Dataset
20 rows, 3 features:
- Study_Hours
- Sleep_Hours
- Attendance

Target:
- Exam_Score

## Model Used
Linear Regression

## Results
MAE:0.9390093654838516
R2 Score: 0.9970604987119472

## Conclusion
Study hours and attendance are the most important factors influencing exam performance.
